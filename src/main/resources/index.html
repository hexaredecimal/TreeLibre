Tree Growth Simulation — Slider Reference

This wiki describes the purpose and behavior of each slider used in the procedural tree generation simulation.

Overview

This simulation models a procedural tree that grows by recursively splitting branches based on probabilistic parameters. The sliders let you adjust biological and environmental factors that affect how the tree splits, bends, and sways with wind. Adjusting them dynamically updates the visual appearance of the tree in real time.

Slider Descriptions

Mean Mass Splitting (f)

Controls how evenly branch thickness is split between two new branches. Higher values produce one dominant branch and one smaller one, while values near 0.5 create more symmetrical branching.

Std Mass Splitting (f)

Sets the randomness of the mass split between branches. Increasing this adds variability to how branch thickness divides at each split, giving a more chaotic or natural look.

Mean Branching Angle (Δθ)

Determines the average angle between child branches. Larger angles produce wider, more open trees; smaller angles yield tighter, narrower growth patterns.

Std Branching Angle (Δθ)

Adds randomness to the branching angles. A higher standard deviation makes each branch deviate more unpredictably, resulting in an irregular, more organic shape.

Branch Length Scaling (γ)

Controls how branch length decreases as the tree splits. A smaller value causes branches to shrink rapidly, while a larger one keeps them longer and fuller.

Root Length

Specifies the initial length of the tree's root segment. Larger values make the base trunk longer, affecting overall tree size and reach.

Root Thickness

Defines the starting thickness (diameter) of the tree trunk. Thicker roots produce a sturdier, broader tree base.

Leaf Cutoff

Determines the minimum branch thickness below which leaves start forming. Lower values create denser foliage; higher values reduce leaf count and emphasize branch structure.

Wind Strength

Controls how strongly the wind affects the tree’s motion. Increasing this makes the tree sway more dramatically. Combine with mouseWind (toggle with ENTER) to add mouse-controlled wind direction.

Interaction

  • Spacebar: Resample the tree with the current slider settings.
  • Enter: Toggle mouse-controlled wind direction on or off.

Notes

The parameters together influence how the tree evolves visually — the balance between mass splitting and branch scaling controls structural realism, while angle deviations and wind effects add motion and organic irregularity.
