This document describes the purpose and behavior of each slider used in the procedural tree generation simulation.

Overview

This simulation models a procedural tree that grows by recursively splitting branches based on probabilistic parameters. The sliders let you adjust biological and environmental factors that affect how the tree splits, bends, and sways with wind. Adjusting them dynamically updates the visual appearance of the tree in real time.

Slider Descriptions

Mean Mass Splitting (f)

Controls how evenly branch thickness is split between two new branches. Higher values produce one dominant branch and one smaller one, while values near 0.5 create more symmetrical branching.

Std Mass Splitting (f)

Sets the randomness of the mass split between branches. Increasing this adds variability to how branch thickness divides at each split, giving a more chaotic or natural look.

Tree Properties

Trunk Length

Specifies the initial length of the tree's trunk segment. Larger values make the base trunk longer, affecting overall tree size and reach.

Trunk Thickness

Defines the starting thickness (diameter) of the tree trunk. Thicker trunks produce a sturdier, broader tree base.

Leaf Properties

Leaf Cutoff

Determines the minimum branch thickness below which leaves start forming. Lower values create denser foliage; higher values reduce leaf count and emphasize branch structure.

Leaf Density

Specifies the percentage of leaves to cover the tree with. Lower values reduce leaf count; higher values create a fuller canopy.

Leaf Height

Sets the vertical size of each leaf.

Leaf Width

Sets the horizontal width of each leaf.

Flower Properties

Flower Density

Adds flowers to the tree, specified by the amount set.

Wind Properties

Wind Strength

Controls how strongly the wind affects the treeâ€™s motion. Increasing this makes the tree sway more dramatically.

Wind Speed

Determines how quickly wind oscillations occur, influencing how fast branches sway.

Extended Controls

Trunk Height Multiplier

Adds a multiplier to the trunk height to create variation in the overall size of the tree.

Trunk Width Multiplier

Adds a multiplier to the trunk width to vary trunk proportions dynamically.

Branch Length Multiplier

Controls the overall branch length scaling under strict simulation conditions.

Taper Effect

Controls how leaf or branch size gradually decreases toward the tips, producing a more natural shape.
